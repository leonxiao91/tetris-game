# 🎮 Tetris Game

A modern, neon-styled Tetris game built with vanilla JavaScript.

## Play Online

The game will be available at: **https://leonxiao91.github.io/tetris-game**

## How to Enable GitHub Pages

1. Go to: https://github.com/leonxiao91/tetris-game/settings/pages
2. Under "Build and deployment", select:
   - **Source**: Deploy from a branch
   - **Branch**: main
   - **Folder**: / (root)
3. Click Save
4. Wait ~1-2 minutes for deployment

## Controls

- **← →** : Move piece
- **↓** : Soft drop (faster)
- **↑** : Rotate
- **Space** : Hard drop
- **P** : Pause/Resume

## Features

- ✅ Classic 7 Tetris pieces
- ✅ Score tracking
- ✅ Level system (increases every 10 lines)
- ✅ Next piece preview
- ✅ Game over detection
- ✅ Pause functionality
- ✅ Beautiful dark theme with neon accents

## Tech Stack

- Pure HTML5/CSS3/JavaScript
- No external dependencies
- Responsive canvas rendering
